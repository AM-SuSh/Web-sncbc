核心价值

Core Values

农业可持续发展

通过使用中药渣化肥，可提高土地的有机质含量，特别是位于中国东北地区的黑土地，有效改善土壤结构，提升作物产量和品质。

推动地方经济发展

中药渣有机肥项目在振兴东北乡村经济中不仅为当地农业的可持续发展提供了支持，还促进了当地农村地区的经济多元化发展。

升级整合产业链

与中药材种植基地、中药加工企业、农业合作社等建立长期合作关系，形成从原料供应到产品销售的全产业链管理。

环境保护与资源循环利用

通过中药渣堆肥化处理，大大提高废弃物占用空间利用率，同时也有效减少了化肥生产过程中的能源消耗和污染物排放。

有机肥技术创新

项目采用微生物降解技术，研发新型微生物菌种等技术创新，不仅提高了中药渣的降解效率，同时提升了产品的质量，为项目带来额外的竞争优势。

弘扬传统中草药文化

项目通过中药渣的高值利用，提升了中药材的附加值。这不仅能够提升中药材的经济价值，还能够弘扬传统中草药文化，使其在现代社会中得到更广泛的应用和认可。
